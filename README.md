# NULL//SHIFT — Online Prototype

AIがプレイヤーを複製して紛れ込む、2Dオンライン疑似人狼サバイバルです。

## 起動

```bash
npm start
```

ブラウザで `http://localhost:3000` を開きます。同じネットワークの端末からは `http://<起動PCのIP>:3000` へ接続できます。インターネット公開時は、HTTPS対応のリバースプロキシ（Caddy / nginx等）越しにポート3000へ転送してください。

## 遊び方

1. 名前と外見を作る
2. 「ルームを作る」で6桁コードを発行、またはコードを入力して参加
3. WASD / 矢印で移動、Eで端末タスク、Spaceでパルス（敵を一時停止）
4. 全端末を復旧して脱出する。誰かの姿をコピーした「MIMIC」に注意
5. VでボイスON/OFF、Enterでチャット

## 実装済み

- Node.js標準機能のみで動くHTTP/WebSocketサーバー
- ルームコード式オンライン同期（最大8人）
- Canvas 2Dマップ、衝突、タスク、HP、観戦、勝敗
- プレイヤー外見メイキング（色・髪・アクセサリー）
- サーバー側AI MIMIC（プレイヤーを複製・追跡・攻撃・擬態チャット）
- WebRTCメッシュ型ボイスチャット（マイク許可制）
- テキストチャットのフォールバック

## 注意

- ボイスはWebRTCの仕様上、公開環境ではHTTPSが必要です（localhostは例外）。
- NAT環境によってはTURNサーバーがないと音声接続できない場合があります。テキストチャットは常に利用できます。
- 現段階は遊べるプロトタイプです。本番運用には認証、永続化、レート制限、TURN、監視、プロセスマネージャーを追加してください。
